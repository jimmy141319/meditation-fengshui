# Shichen Calm | Product Report (English)

## Positioning
- Type: In-car calm music + meditation app
- Platform: iPad / iPhone Web App / PWA
- Price: NT$99 one-time purchase
- Language: Chinese / English
- Audio strategy: original generated soundscapes to avoid direct copyright risk from commercial music

## Included in this version
1. Auto-switch by time segment on app entry
2. Chinese / English toggle
3. Meditation mode and 90-second reset breathing flow
4. Base-tone (Hz) and pulse (BPM) labels
5. PWA setup for iPad home screen
6. One-time purchase wording (NT$99)

## Copyright strategy
- The app currently uses original generated soundscapes
- It can run without external playlists
- It can serve as the base version for future AI music generation
- It does not claim to ship with third-party copyrighted songs

## Frequency labeling principle
- The displayed Hz values are creative base tones
- They help differentiate moods and time windows
- They are not medical claims, diagnoses, or treatment guarantees

## Suggested next version
- Add more soundscape layers
- Add Driver Mode / Passenger Mode
- Add bilingual voice guidance
- Create commercial UI screenshots and a landing page

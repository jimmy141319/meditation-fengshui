# 時辰安穩｜產品報告（中文）

## 產品定位
- 類型：車內平穩音樂 + 冥想 App
- 平台：iPad / iPhone Web App / PWA
- 售價：NT$99 一次買斷
- 語言：中文 / 英文雙版本
- 音訊策略：以原創生成聲景為主，避免直接使用受版權保護的商業音樂

## 這版已加入
1. 一進入 App 自動依照時辰切換模式
2. 中文 / 英文切換
3. 冥想模式與 90 秒呼吸拉回
4. 基準音頻（Base Tone）與節奏（BPM）標示
5. PWA 設定，可加入 iPad 主畫面
6. 一次買斷 NT$99 的產品文案

## 版權策略
- 目前 App 內聲音為原創生成聲景
- 不依賴外部歌單即可運作
- 可作為後續 AI 音樂生成的基礎版本
- 不宣稱使用他人歌曲或既有商用音檔

## 頻率標示原則
- 畫面中顯示的 Hz 為創作用基準音頻 / Base Tone
- 用於建立不同時段與情緒的聲景差異
- 不作醫療療效保證或診斷宣稱

## 建議下一版
- 加入更多聲景層次
- 加入司機模式 / 乘客模式
- 加入中英文語音引導
- 製作正式商業版 UI 截圖與 App Landing Page
